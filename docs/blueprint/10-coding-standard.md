# 10. Coding Standard

- Struktur folder (mengikuti default framework yang dipilih, jangan dipaksa custom biar cepat)
- Naming convention: tabel snake_case, model PascalCase, dst. sesuai framework
- Format response API kalau backend dipisah (status, message, data)
- Git flow: branch `main`/`develop`/`feature/*`, commit convention (misal Conventional Commits)
- Linting/formatting tool (Pint untuk Laravel, ESLint+Prettier untuk JS)
- Aturan penulisan query (hindari N+1, pakai eager loading dsb — relevan karena banyak tabel relasi tahun ajaran)
