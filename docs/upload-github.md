# Upload GitHub

Proyek ini telah berhasil diunggah ke GitHub pada tahap penyelesaian persiapan Tahap 1, tepat sebelum memulai eksekusi Tahap 2.

**Informasi Repository:**
- URL: `https://github.com/sutrisnoetseul21/absensi-barcode`
- Branch Utama: `main`

## Langkah-langkah yang telah dilakukan:
1. Inisialisasi Git di repositori lokal.
2. Penambahan remote origin SSH (`git@github.com:sutrisnoetseul21/absensi-barcode.git`).
3. Commit awal (Initial commit - Tahap 1) dengan konfigurasi username dan email.
4. Push ke branch `main`.

File ini dibuat sebagai penanda bahwa kode sumber telah diamankan dan direkam riwayatnya pada repositori di atas.
